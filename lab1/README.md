# Amazon Bedrock を使用してキャッチコピー生成を最適化する

<!--include:Header-->

## ラボの概要

あなたはマーケティングエグゼクティブであり、クライアントはエコロジーを主要なセールスポイントとするコーヒー会社です。コーヒーショップのクライアント向けにキャンペーンのキャッチコピーと包括的な広告提案書を作成するよう依頼されました。

このラボでは、Amazon Bedrock と Amazon Nova Pro を使用してマーケティング代理店向けのキャッチコピーを生成します。基本的なプロンプトから始めて、プロンプトエンジニアリングの手法を用いて段階的に改良し、より効果的でカスタマイズされたキャッチコピーを作成します。

### 目標

このラボを修了すると、以下のことができるようになります。

* 基本的な生成 AI のプロンプト手法を用いてキャッチコピーを生成する。
* クライアントに関する詳細なコンテキストを提供してプロンプトを改良する。
* キャッチコピーのトーンとスタイルを指定する。
* フューショットプロンプティングの手法を使用する。
* 特定の指示でプロンプトを最適化する。
* 広告提案書を作成する。

### このラボで使用する AWS のサービス

#### Amazon Bedrock

Amazon Bedrock は、主要な AI 企業が提供する高パフォーマンスの基盤モデル (FM) を 1 つの API で選択して利用できるフルマネージドサービスです。セキュリティ、プライバシー、責任ある AI を備えた生成 AI アプリケーションを構築するための幅広い機能が用意されています。

## タスク 1: キャッチコピーを作成して改良する

このタスクでは、コーヒーショップの基本的なキャッチコピーを作成し、反復的に改良します。

### タスク 1.1: このラボで使用するモデルを確認する (オプション)

このラボでは Amazon Nova Pro モデルを使用します。

1. AWS マネジメントコンソールの検索ボックスで **`Amazon Bedrock`** を検索してクリックします。

1. Amazon Bedrock コンソールで、左側のメニューパネルがまだ表示されていない場合は展開し、[**モデルカタログ**] を選択します。

1. [**モデルのフィルター**] をクリックして [**プロバイダー**] を選択して、さらに [**プロバイダー = amazon**] を選択します。

1. 表示された基盤モデルのパネルから **`Nova Pro`** をクリックすると、その詳細が表示されます。

### タスク 1.2: 基本的なキャッチコピーの生成

Amazon Bedrock のチャット/テキストプレイグラウンドで、プロンプトを実行して基本的なキャッチコピーを作成します。

1. Amazon Bedrock コンソールの [**テスト**] で、[**Playground**] を選択します。

1. [**モデルを選択**] をクリックします。

1. [**1. カテゴリ**] で [**Amazon**] を選択し、[**2. モデル**] で[**Nova Pro**] を選択します。

1. <span style="ssb_orange_oval">[**適用**]</span> をクリックします。

1. [**モード**] から、[**シングルプロンプト**] を選択します。

1. ページ下部のプロンプトの入力エリアで以下を貼り付けます。

    ```text
    コーヒーショップの広告キャッチコピーを5つ生成してください。
    ```

1. <span style="ssb_orange_oval"><i class="fa fa-play"></i> [**実行**]</span> をクリックします。

    <i aria-hidden="true" class="fas fa-clipboard-check" style="color:#18ab4b"></i> **想定される出力:** さまざまな応答が考えられますが、以下のような 5 つのキャッチコピーが含まれる可能性があります。

    ```text
    コーヒーショップの広告キャッチコピーを5つご紹介します。

    1. 目覚めよ、素晴らしさの香りとともに。
    2. 一杯ずつ、完璧に淹れる。
    3. 風味豊かな幸せであなたの一日を満たそう。
    4. 一口ごとが至福のひととき。
    5. 一杯のコーヒーで、笑顔を届ける。
    ```

     * <i aria-hidden="true" class="fas fa-sticky-note" style="color:#563377"></i> **注意:** これらの演習ではシングルプロンプトを使用しているため、プロンプトごとにテキスト領域から前の質問/回答を消去して、現在の質問のみに基づいて応答するようモデルの焦点を定める必要があります。テキスト領域を消去するには、テキスト領域の右上にある [**Clear**] ボタン (ほうき/ブラシアイコン) をクリックします。

### タスク 1.3: 推論パラメータをテストする

テキスト領域の下にあるモデルタイトルの左側に、[***Configurations***] という推論パラメータのペインが表示されます。この [***Configurations***] ペインにある設定から、出力を変えるよう調整することができます。**Temperature** パラメータと **Top** P パラメータは、モデル応答の創造性や決定性を制御します。デフォルトでは、Temperature スライダーと Top P スライダーは右端に設定されているため、よりクリエイティブなキャッチコピーが生成されます。

<i aria-hidden="true" class="fas fa-sticky-note" style="color:#563377"></i> **注意:** [**Configurations**] ペインが表示されていない場合は、[**Mode**] 選択の下にあるアイコンをクリックします。

1. よりクリエイティブなキャッチコピーを生成します。前のタスクと同じプロンプトを、デフォルトの **温度** と **トップ P** (スライダーを右端に設定) でさらに数回試します。

    <i aria-hidden="true" class="fas fa-sticky-note" style="color:#563377"></i> **注意:** モデル出力では、実行するたびに異なるキャッチコピーや、よりクリエイティブでユニークなキャッチコピーが生成される場合があります。

2. **温度** スライダーと **トップ P** スライダーを左端に調整し、同じプロンプトを再実行します。

    <i aria-hidden="true" class="fas fa-sticky-note" style="color:#563377"></i> **注意:** モデルの出力に注目してください。同じプロンプトを数回実行すると、毎回よく似たキャッチコピーが生成される可能性があります。

### タスク 1.4: LLM ペルソナを調整して出力を制御する

プロンプトでは、特定の性格、キャラクター、役割を採用するよう指示することで、出力のトーンやスタイルを制御することができます。

1. 特定のペルソナを定義する以下のプロンプトを実行して、キャッチコピーを生成します。
    ```text
    あなたはマーケティングエグゼクティブです。
    コーヒーショップの広告キャッチコピーを5つ生成してください。
    ```

    * <i aria-hidden="true" class="fas fa-sticky-note" style="color:#563377"></i> **注意:** このペルソナのモデルが作成したキャッチコピーに注目してください。

1. 以下の編集済みプロンプトを実行して、別のペルソナを指定します。
    ```text
    あなたはマディソン街のトップマーケティング会社で長年の経験を持つ、非常に経験豊富なマーケティングエグゼクティブです。キャッチーで説得力のあるキャッチコピーを生み出す専門知識と実績があります。あなたの仕事は、コーヒーショップの広告キャッチコピーを5つ生成することです。
    ```
    * <i aria-hidden="true" class="fas fa-sticky-note" style="color:#563377"></i> **注意:** このペルソナで作成されたキャッチコピーからは、先ほどのものより洗練され考え抜かれた印象を受けます。

    * <i aria-hidden="true" class="fas fa-sticky-note" style="color:#563377"></i> **注意:** LLM に「ペルソナ」を提示し、そのペルソナが「経験豊富」であることを指定する場合、その職務において豊富な経験を持つ人物の知識や視点に沿った応答を生成するようモデルを導くための追加のコンテキストを提供します。これにより、特定の状況に沿った、よりニュアンス豊かで情報に基づいた応答が得られる可能性があります。

### タスク 1.5: プロンプトにコンテキストを追加する

1. プロンプトを以下のように修正するか、編集したプロンプトをテキストフィールドにコピーします。


1. <span style="ssb_orange_oval"><i class="fa fa-play"></i> Run</span> をクリックします。

    <i aria-hidden="true" class="fas fa-clipboard-check" style="color:#18ab4b"></i> **想定される出力:** さまざまな応答が考えられますが、以下のようなものになる可能性があります。

    ```text
    2. 小さな農園、大きな風味: 持続可能な違いを味わおう。
    ```

## タスク 2: 広告提案書を作成する

この最後の演習では、前のステップで生成したキャッチコピーを使用して広告提案書を作成します。この文書ではキャッチコピーを提示し、それぞれのキャッチコピーについて簡潔な説明を加え、クライアントのブランドやターゲットオーディエンスにどのように合致するかを示します。出力にはマークダウン形式のテキストを使用し、さまざまなフォントサイズを用いたより洗練された出力を作成します。マークダウン形式で出力を作成するよう LLM に指示することができます。

<i aria-hidden="true" class="fas fa-sticky-note" style="color:#563377"></i> **注意:** マークダウンの詳細については、[マークダウンガイドの基本的な構文](https://www.markdownguide.org/basic-syntax/) に関するドキュメントページを参照してください。

### タスク 2.1: 最終版広告提案書のプロンプトを作成する

1. 最終版広告提案書のプロンプトを以下のように作成します。

    ```text
    小規模農園から直接豆を仕入れている、高級でエコフレンドリーなコーヒーショップの広告提案書を作成してください。文書には以下のセクションを含めてください。

    1. コーヒーショップの持続可能性への取り組みと職人的なコーヒー製造方法について説明する導入段落。

    2. 5つのキャッチコピーセクション。各セクションには以下を含めてください。
    - 「職人的な」「持続可能な」「エコフレンドリーな」「倫理的な」などのキーワードを取り入れた、ウィットに富み洗練されたキャッチコピー（5〜6語）。可能な限り言葉遊び、頭韻法、韻を使用してください。
    - そのキャッチコピーがコーヒーショップの価値観をどのように捉え、環境意識の高いミレニアル世代の顧客にどのように訴求するかを説明する2〜3文の解説。

    3. キャッチコピー全体が、高品質なコーヒーを称えながら、ブランドをエコ意識の高いものとしてどのように位置づけているかをまとめた結びの段落。

    全体を通して、洗練され、クリエイティブで、魅力的なトーンにしてください。ターゲットであるミレニアル世代に向けて、カフェのユニークなセールスポイントを強調することに重点を置いてください。

    以下の例に従って、マークダウン形式のテキストフォーマットを追加してください。
    
    # エコフレンドリーなコーヒーショップ: 広告提案書
    ## キャッチコピー 1: 「サンプルキャッチコピー 1」
    このキャッチコピーは...
    ... 
    ## 結びの段落
    これらのキャッチコピーを通じて...
    ```

1. <span style="ssb_orange_oval"><i class="fa fa-play"></i> Run</span> をクリックします。

*	<i aria-hidden="true" class="fas fa-clipboard-check" style="color:#18ab4b"></i> **想定される出力:** さまざまな応答が考えられますが、以下のセクションが含まれているはずです。
    * 導入段落
    * 5 つのキャッチコピーセクション。各セクションにはキャッチコピーとキャッチコピーの説明が含まれています。
    * 結びの段落

<i aria-hidden="true" class="far fa-thumbs-up" style="color:#008296"></i> **タスク完了:** プロンプト手法を反復的に使用し、広告提案書を作成して改良できました。
